# 传感器封装

> 日期：2026-05-14

---

## 一、概述

项目中传感器代码统一封装为 `bsp_xxx.h` + `bsp_xxx.c` 驱动对，遵循统一 API 模式。

**已封装的传感器：**

| 头文件 | 传感器 | 接口 | 引脚/地址 |
|--------|--------|------|-----------|
| `bsp_sht20.h` | SHT20 温湿度 | I2C1 | 0x40 |
| `bsp_ap3216c.h` | AP3216C 环境光/接近 | I2C1 | 0x1E |
| `bsp_scd41.h` | SCD41 CO2 | I2C1 | 0x62 |
| `bsp_mq2.h` | MQ-2 烟雾 | GPIO | PB8 |
| `bsp_mq3.h` | MQ-3 酒精 | GPIO | PB9 |
| `bsp_motor.h` | 震动马达 | GPIO | PC7 |
| `bsp_max30102.h` | MAX30102 心率血氧 | I2C1 | 0x57 |
| `bsp_ili9341_4line.h` | ILI9341 LCD | SPI | — |
| `bsp_ft6336.h` | FT6336 触摸屏 | I2C1 | 0x38 |
| `bsp_ospi_w25q128.h` | W25Q128 Flash | OSPI | — |
| `bsp_esp8266.h` | ESP8266 WiFi | UART5 | — |

## 二、如何使用

### 2.1 引入头文件

在 `main.c` 或 `user_app.c` 中 include：

```c
#include "bsp_mq2.h"       // MQ-2 烟雾传感器
#include "bsp_mq3.h"       // MQ-3 酒精传感器
#include "bsp_motor.h"     // 震动马达
#include "bsp_max30102.h"  // MAX30102 心率血氧
```

### 2.2 调用模式

所有传感器遵循统一的四步使用模式：

```c
// 1. 初始化
BSP_MQ2_Init();
BSP_MOTOR_Init();

// 2. 读取状态（按需）
uint8_t smoke = BSP_MQ2_Read();      // 0=报警, 1=正常
uint8_t alcohol = BSP_MQ3_Read();    // 0=报警, 1=正常

// 3. 控制执行器（按需）
BSP_MOTOR_On();                       // 开启马达
HAL_Delay(3000);                      // 震动 3 秒
BSP_MOTOR_Off();                      // 关闭马达

// 4. 测试函数（阻塞运行，仅调试用）
BSP_MQ2_Test();   // MQ-2 完整测试：自检→预热→模拟报警→监控
BSP_MQ3_Test();   // MQ-3 完整测试
BSP_MAX30102_Test();  // MAX30102 测试：初始化→读ID→读温度→读FIFO
BSP_MOTOR_Test();  // 马达自检 2 秒
```

### 2.3 GPIO 传感器使用示例（MQ-2）

```c
// 初始化
BSP_MQ2_Init();

// 监控循环
while(1)
{
    uint8_t smoke = BSP_MQ2_Read();
    if(smoke == 0)
    {
        printf("[MQ-2] Smoke alarm!\r\n");
        BSP_MOTOR_On();
    }
    else
    {
        BSP_MOTOR_Off();
    }
    HAL_Delay(300);
}
```

### 2.4 I2C 传感器使用示例（MAX30102）

```c
// 初始化
BSP_MAX30102_Init();

// 读取芯片温度
float temp;
BSP_MAX30102_ReadTemperature(&temp);
printf("[MAX30102] Temp=%.2f C\r\n", temp);

// 读取 FIFO 数据
uint32_t red, ir;
if(BSP_MAX30102_ReadFIFO(&red, &ir))
{
    printf("[MAX30102] RED=%lu IR=%lu\r\n", red, ir);
}
```

## 三、如何封装

### 3.1 文件命名规范

| 元件类型 | 文件名 | 函数前缀 |
|----------|--------|----------|
| 传感器 | `bsp_xxx.h` / `bsp_xxx.c` | `BSP_XXX_` |
| 执行器 | `bsp_xxx.h` / `bsp_xxx.c` | `BSP_XXX_` |

### 3.2 头文件模板（`bsp_xxx.h`）

```c
/**
  ******************************************************************************
  * @file   bsp_xxx.h
  * @brief  XXX传感器驱动
  ******************************************************************************
  */
#ifndef __BSP_XXX_H__
#define __BSP_XXX_H__

#include "main.h"

// 硬件定义
#define XXX_GPIO_PORT       GPIOB
#define XXX_GPIO_PIN        GPIO_PIN_8
// I2C 设备定义（I2C 传感器用）
// #define XXX_I2C_ADDR     0x57

// 数据结构（按需）
typedef struct
{
    float   value;
    uint8_t status;
} XXX_Data_t;

// API 函数
void     BSP_XXX_Init(void);
uint8_t  BSP_XXX_Read(void);
void     BSP_XXX_Test(void);

#endif /* __BSP_XXX_H__ */
```

### 3.3 GPIO 传感器封装示例（MQ-2）

```c
/**
  ******************************************************************************
  * @file   bsp_mq2.c
  * @brief  MQ-2烟雾传感器驱动
  ******************************************************************************
  */
#include "bsp_mq2.h"

/*
**********************************************************************
* @fun     :BSP_MQ2_Init
* @brief   :初始化MQ-2传感器GPIO (PB8 输入上拉)
* @param   :none
* @return  :none
**********************************************************************
*/
void BSP_MQ2_Init(void)
{
    GPIO_InitTypeDef g = {0};
    g.Pin = MQ2_GPIO_PIN;
    g.Mode = GPIO_MODE_INPUT;
    g.Pull = GPIO_PULLUP;
    HAL_GPIO_Init(MQ2_GPIO_PORT, &g);
}

/*
**********************************************************************
* @fun     :BSP_MQ2_Read
* @brief   :读取MQ-2传感器状态
* @param   :none
* @return  :0=检测到气体(报警), 1=正常
**********************************************************************
*/
uint8_t BSP_MQ2_Read(void)
{
    return (HAL_GPIO_ReadPin(MQ2_GPIO_PORT, MQ2_GPIO_PIN) == GPIO_PIN_RESET) ? 0 : 1;
}
```

### 3.4 GPIO 执行器封装示例（震动马达）

```c
/**
  ******************************************************************************
  * @file   bsp_motor.c
  * @brief  震动马达驱动 (PC7)
  ******************************************************************************
  */
#include "bsp_motor.h"

void BSP_MOTOR_Init(void)
{
    BSP_MOTOR_Off();    // 确保初始关闭
}

void BSP_MOTOR_On(void)
{
    HAL_GPIO_WritePin(MOTOR_GPIO_PORT, MOTOR_GPIO_PIN, GPIO_PIN_SET);
}

void BSP_MOTOR_Off(void)
{
    HAL_GPIO_WritePin(MOTOR_GPIO_PORT, MOTOR_GPIO_PIN, GPIO_PIN_RESET);
}

void BSP_MOTOR_Toggle(void)
{
    HAL_GPIO_TogglePin(MOTOR_GPIO_PORT, MOTOR_GPIO_PIN);
}
```

### 3.5 I2C 传感器封装示例（MAX30102 包装）

已有外部驱动库时，BSP 层做薄包装统一接口：

```c
/**
  ******************************************************************************
  * @file   bsp_max30102.c
  * @brief  MAX30102心率血氧传感器驱动 (I2C1 0x57)
  ******************************************************************************
  */
#include "bsp_max30102.h"
#include "i2c.h"

void BSP_MAX30102_Init(void)
{
    maxim_max30102_init();    // 调用底层驱动
}

uint8_t BSP_MAX30102_ReadID(void)
{
    uint8_t id = 0;
    maxim_max30102_read_reg(REG_PART_ID, &id);
    return id;
}

uint8_t BSP_MAX30102_ReadFIFO(uint32_t *pRed, uint32_t *pIR)
{
    uint8_t status = 0;
    maxim_max30102_read_reg(REG_INTR_STATUS_1, &status);
    if (status & 0xC0)
    {
        maxim_max30102_read_fifo(pRed, pIR);
        return 1;
    }
    return 0;
}

void BSP_MAX30102_ReadTemperature(float *pTemp)
{
    int8_t  temp_int = 0;
    uint8_t temp_frac = 0;
    maxim_max30102_read_temperature(&temp_int, &temp_frac);
    *pTemp = (float)temp_int + ((float)temp_frac) / 16.0f;
}

void BSP_MAX30102_Reset(void)
{
    maxim_max30102_reset();
}
```

### 3.6 Test 函数模式

每个驱动应提供一个 `Test()` 函数用于独立调试，包含必要的自检流程：

```c
void BSP_XXX_Test(void)
{
    printf("[XXX] === Sensor Test ===\r\n");

    // 1. 初始化
    printf("[XXX] Initializing...\r\n");
    BSP_XXX_Init();

    // 2. 读取标识 / 自检
    uint8_t id = BSP_XXX_ReadID();
    printf("[XXX] ID: 0x%02X\r\n", id);

    // 3. 读取数据
    printf("[XXX] Reading data...\r\n");
    // ... 读取并打印 ...

    printf("[XXX] Test done.\r\n");
}
```

## 四、封装原则

| 原则 | 说明 |
|------|------|
| **统一命名** | 头文件 `bsp_xxx.h`，函数 `BSP_XXX_` 前缀 |
| **最少 API** | 仅暴露 Init / Read / GetData / Test，不暴露寄存器细节 |
| **自检能力** | 每个驱动提供 `Test()` 函数，可在 main.c 中直接调用验证硬件 |
| **隔离依赖** | 驱动只依赖 `main.h`，不依赖 TouchGFX 或任务调度器 |
| **注释规范** | 仿照 STM32 HAL 库风格，函数块标注功能、参数、返回值 |
| **薄包装** | 已有外部驱动库时做薄包装，不重复造轮子 |

## 五、main.c 初始化规范

所有传感器初始化集中在 `USER CODE BEGIN 2` 区域，格式统一：

```c
/* USER CODE BEGIN 2 */
// 清空任务列表
for(gTaskIndex = 0; gTaskIndex < OS_TASKLISTCNT; gTaskIndex++)
    g_OSTsakList[gTaskIndex] = NULL;

// NOR Flash 初始化
OSPI_W25Qxx_Init();
OSPI_W25Qxx_mmap();

//=== 传感器初始化 ===
BSP_MQ2_Init();
BSP_MQ3_Init();
BSP_MOTOR_Init();
//=== 初始化完成 ===

// 测试调用（调试时取消注释，阻塞运行）：
// BSP_MQ2_Test();       // MQ-2烟雾传感器测试
// BSP_MQ3_Test();       // MQ-3酒精传感器测试
// BSP_MAX30102_Test();  // MAX30102心率血氧测试
// BSP_MOTOR_Test();     // 马达自检

// NFC 初始化
NFC_Init(&nfc, &huart4, &huart1, nfc_rx_pool, NFC_BUF_SIZE);
// ...
/* USER CODE END 2 */
```
